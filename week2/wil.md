# 2주차 WIL
## 1. 학습한 내용
### 스프링 계층형 아키텍쳐
- Browser : UI 제공, user의 액션을 HTTP 요청으로 바꿈
- Controller : browser가 보낸 데이터를 읽고 service를 호출
- Service : controller부터 데이터를 전달받아 로직 수행
- DAO : DB 관리인
- DB : 데이터를 저장, 결과 반환
- DTO : 소통 목적에 맞는 필요한 정보만 전달

### 스프링 빈 & 의존성 주입
- Spring : 대표적인 java 백엔드 어플리케이션 프레임워크
- Spring Boot : Spring 쉽게 쓸 수 있게 해주는 도구
- Spring bean : 어플리케이션 전역에서 사용할 공용 객체

## 느낀점
- 용어나 개념에서는 어려움을 느끼지 못하였으나 코드를 따라서 구현해 보는 부분에서 어려움을 느꼈다. 단순 코딩 뿐만 아니라 intelliJ같은 IDE에 익숙해지는 것도 내가 코딩을 멀리 하게 된 이유 중 하나였다. 언제까지나 안할 수는 없어 오랜만에 이것저것 건드려보았는데 익숙해지려면 시간이 좀 걸리겠다 싶었다. 코드 내에서도 이해할 수 없는 부분이 많았다. 알려주시는 멘토? 분은 어떻게 저리 편하게 입력을 할까 생각을 했다. 프로젝트에 참여하고 싶은 마음이 큰데 더 열심히 해야겠다.
